About MediQR

MediQR is a platform for convenient and secure medical record sharing. The QR code for each patient can only be decoded with the patient's password. The Medical profile on the left cannot be shown to any one except for the patient himself/herself. Even if the QR code is leaked, the profile will still be safe and remain crypted.
